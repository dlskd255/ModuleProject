걸어서 세계속으로 설명서

git에 있는 파일 zip파일로 다운로드하고 압축 풀기

미니콘다 프롬프트에서 파이썬 3.8 환경을 만든 후 다운로드하고 실행

보정과정(위 숫자가 300/300이 될 때까지 대기)
명령어 목록: 보정, 그림 시작, 그림 종료, 지워 줘, 사진

녹음이 시작되는 소리가 나면 원하는 지역(예: 경복궁)을 말해서 위치로 이동

돌아보고 싶은 방향으로 머리 회전(회전 시 해당 방향으로 시선 이동)

"그림 시작" 명령어 실행 시 캠 위에 그리기 시작

"사진" 명령어 실행 시 화면 캡쳐 및 병합
(명령어 "사진" 또는 손흥민 포즈를 취할 시 사진을 찍을 수 있음)

"지워 줘" 명령어 실행 시 캠 위에 그렸던 그림 삭제
